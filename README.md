# HTML Basico - sua primeira pagina

_Crie um documento HTML do zero e pratique as principais tags._

## Boas-vindas

- **Para quem e**: Estudantes de graduacao que estao iniciando em desenvolvimento web.
- **O que voce vai aprender**: Estrutura basica de HTML, `title`, `h1`, paragrafo, link, imagem e `footer`.
- **O que voce vai construir**: Um arquivo `index.html` completo, sem CSS.
- **Pre-requisitos**:
  - Conta no GitHub
  - Conhecimento basico de Git

- **Tempo estimado**: Este exercicio leva cerca de 20 a 30 minutos.

Neste exercicio, voce vai:

1. Criar o arquivo `index.html`.
1. Montar a estrutura basica do HTML.
1. Definir um `title` para a pagina.
1. Adicionar um paragrafo com "Ol√° HTML".
1. Adicionar uma tag `h1` com "Seja bem-vindo ao meu site!".
1. Criar um link para `atitus.edu.br`.
1. Inserir uma imagem do `placehold.co` com atributo `alt`.
1. Finalizar com um `footer`.


### Como iniciar este exercicio

Copie este exercicio para sua conta, aguarde **cerca de 20 segundos** para preparar a primeira etapa e **atualize a pagina**.
